# go_course

Минимальная настройка для разработки на Go.

Быстрый старт:

- Установите Go (версия 1.21+).
- Откройте папку в VS Code и установите рекомендованное расширение `golang.go`.
- Собрать проект: `go build ./...`
- Запустить пример: `go run .`
- Запустить тесты: `go test ./...`

VS Code:

- В `Run and Debug` выберите конфигурацию `Launch Package` для отладки (Delve).
- В `Terminal -> Run Task...` доступны задачи `Go: Build`, `Go: Run`, `Go: Test`.
